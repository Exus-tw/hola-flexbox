Box Model

  • Separa estilo da marcação
  • Muito utilizado hoje em dia
            
              .button {
                display: block;
                float: right;
                width: 200px;
                height: 20px;

                border: 1px solid #333;
              }

              .footer {
                position: fixed;
                bottom: 0;
                margin-top: 10px;
              }

Problemas

  • Alinhar items verticalmente
  • Clearfix hell
  • Altura altomática
  • Boxes com a mesma altura
  • Posicionar items na mesma linha (float, inline ou inline-block?)
             
               .clearfix {
                 content: "";
                 display: table;
                 clear: both;
               }

               .inline {
                 display: inline;
                 float: right;
                 display: inline-block;
               }

               .align-middle {
                 vertical-align: middle;
                 display: table-cell;
               }

¿Cómo usar Flexbox?

Modelo Contenedor/hijo(s):

Layout flex

Componente flex

Propiedades de flexbox:

  • flex-direction: layout content horizontally and vertically
  • order: rearrange flexbox children
  • alignment: flexbox children
  • flex-basis: dimensions on children
  • flex-shrink & flex-grow: resize children correctly
  • flex shorthand: Combining sizing properties
  • flex-wrap and align-content: grid
